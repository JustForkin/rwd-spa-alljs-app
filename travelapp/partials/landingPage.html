Amazing India SPA

Visit some of the exciting destinations in India. The idea behind this site was to create an end to end java script based SPA. Tools and Frameworks used AngularJS, Bootstrap , Node.js , MongoDB for Persistence , Social Buttons for Twitter Bootstrap, JQuery One Page Navigation Plugin
